# Salve o Brasil
O Brasil precisa da sua ajuda, se não todo hemisfério sul será destruido por uma tragédia nuclear. A usina de Angra IV está em risco de colapso, pois o sistema foi invadido e completamente desconfigurado, usando uma quantidade enorme de energia nuclear. Seu papel é ajudar a restaurar o sistema e evitar uma tragédia nuclear, em um período limitado de tempo.

## Como rodar o projeto
1. Instale as dependências:
   `npm install`
2. Execute o aplicativo:
   `npm run dev`
3. Abra em seu navegador e aproveite!
